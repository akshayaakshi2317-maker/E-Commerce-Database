# Nykaa – Beauty Store

## Overview
Nykaa – Beauty Store is an online beauty and personal care shopping website that enables customers to explore, purchase, and manage beauty products with a secure and user-friendly shopping experience.

## Problem Statement
   With the increasing demand for online shopping, customers expect quick and convenient access to beauty products. However, traditional shopping requires visiting multiple stores, which consumes time and effort. Customers may also face limited product availability, difficulty in comparing products, and lack of detailed product information. This project solves these issues by providing an online platform where customers can easily search, compare, purchase, and track beauty products from anywhere at any time.

## Project Description
This project is an e-commerce web application developed to simplify online beauty shopping. It provides features such as user registration, product browsing, category-wise search, shopping cart, wishlist, secure payment, order tracking, and an admin dashboard for managing products, customers, and orders.
